Frontend
HTML: The standard markup language for creating the structure of web pages.
CSS: Cascading Style Sheets are used for styling and layout of web pages.
JavaScript: A programming language that enables interactive and dynamic content on the client side.
React: A JavaScript library for building user interfaces, particularly for single-page applications.
Backend
Node.js: A JavaScript runtime that allows server-side execution of JavaScript code.
Express.js: A web application framework for Node.js, simplifying the creation of server-side logic
Database
MySQL: An open-source relational database management system for storing and managing data.
MongoDB: A NoSQL database, using JSON-like documents with optional schemas, suitable for flexible data models.
Server
Nginx: A web server that can also be used as a reverse proxy, load balancer, and HTTP cache.
