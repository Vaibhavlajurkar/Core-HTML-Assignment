1. HTMl

    Primary Use cases Key Features/Benefits

    • Compatible with all web browsers

  2. CSS

    Primary Use Cases Key Features/Benefits
    Styling and layout of web pages.
    • Separation of content and presentation.
    • Responsive design capabilities.
    • Flexibility in styling elements.

  3. JavaScript

    Primary Use Cases Key Features/Benefits
    Adding interactivity and dynamic behavior to web pages.
    • Client-side scripting.
    • Asynchronous programming (AJAX)
    • Large ecosystem of libraries and frameworks (e.g., React, Angular, Vue).
